# 🛠️ موقع سباك وكهربائي محمد بالرياض - Landing Page

موقع إلكتروني احترافي متكامل (Landing Page) مصمم خصيصاً لخدمات السباكة والكهرباء المنزلية وخدمات الطوارئ على مدار 24 ساعة في مدينة الرياض. تم بناء الموقع بأحدث معايير تجربة المستخدم (UI/UX) وتحسين محركات البحث المحلية (Local SEO).

## 🚀 المميزات التقنية للموقع

* **تحسين كامل للسيو (SEO Optimized):** استخدام الكلمات المفتاحية الأكثر بحثاً في الرياض لضمان تصدر نتائج البحث.
* **بيانات منظمة (Schema.org Integration):** دمج كود مخصص لنشاط `HomeAndConstructionBusiness` لمساعدة عناكب جوجل في فهم خدمات الموقع، أوقات العمل، ونطاق التغطية الجغرافية بدقة.
* **تصميم متجاوب (Fully Responsive):** متوافق تماماً مع جميع الشاشات (جوال، أجهزة لوحية، حواسيب محمول ومكتبية).
* **أزرار إجراء سريعة (Click-to-Action):** أزرار عائمة وثابتة للاتصال الهاتفي والمراسلة الفورية عبر الواتساب لرفع معدل التحويل (Conversion Rate).
* **نموذج اتصال تفاعلي:** تحويل مدخلات العميل (الاسم، الجوال، نوع الخدمة، تفاصيل العطل) تلقائياً إلى رسالة واتساب منسقة ومجهزة للإرسال الفوري لمالك الموقع.
* **أداء فائق السرعة:** تم الاعتماد على أكواد نظيفة (Vanilla HTML/CSS/JS) بدون مكتبات ثقيلة لضمان تحميل الموقع في أجزاء من الثانية.

## 🛠️ التكنولوجيات المستخدمة

* **HTML5** - لبناء الهيكل والدلالات البرمجية المهيئة للسيو.
* **CSS3** - للتصميم، التنسيق المتجاوب، والمتغيرات (Custom Properties).
* **JavaScript (ES6)** - لإضافة التفاعلية (قائمة الجوال، قائمة الأسئلة الشائعة الأكورديون، ومعالجة نموذج الاتصال).
* **Google Fonts** - خط "Cairo" المتميز للواجهات الاحترافية العربية.

## 💻 طريقة التشغيل والرفع

الموقع جاهز للعمل مباشرة عبر خدمة **GitHub Pages**:
1. قم برفع ملف `index.html` إلى المستودع (Repository).
2. توجه إلى **Settings** ثم **Pages**.
3. اختر فرع الـ `main` كـ Source واضغط **Save**.
